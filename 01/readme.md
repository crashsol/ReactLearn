# 从零创建项目
 - 全局安装webpack 
   > npm i webpack -g
     npm i webpack-cli -g
 - 搭建项目结构
    - npm init -y 
    - 创建dist和src文件夹
    - 在src创建index.html和main.js
    - 添加webpack
      > yarn add webpack -D
        yarn add webpack-cli -D
        yarn add webpack-dev-server -D
    - 在package.json，scripts中添加 
      > "dev":"webpack-dev-server"
